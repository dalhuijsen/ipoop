JavaTM 2 Platform Security Architecture

This document gives an overview of the motivation of the major security features that are being implemented for the Java 2 SDK, introduces the new classes and their usage, discusses the impact of this new architecture on existing code, and gives thoughts on writing security-sensitive code in the future.
