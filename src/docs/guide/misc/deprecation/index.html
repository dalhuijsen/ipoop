Deprecation of APIs

 Documentation Contents
A deprecated API is an API that we recommend you no longer use, due to improvements in Java. While deprecated APIs are currently still implemented, they may be removed in future implementations, and we recommend using other APIs. The @deprecated tag denotes a deprecated API.
