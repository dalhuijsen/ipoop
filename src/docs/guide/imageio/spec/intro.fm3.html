1.3 Platforms

The API will be a standard part of the "Merlin" release of the Java Platform Standard Edition (J2SE version 1.4). Sun will provide an implementation of the J2SE platform for the Solaris operating environment, Linux, and Microsoft Windows platforms. The availibility of a particular J2SE release on other platforms is determined by the platform vendor.
