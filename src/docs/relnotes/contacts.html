Contacting Java Software

 Documentation Contents

The Java 2 SDK, Standard Edition is a product of Sun Microsystems, Inc. The Java 2 SDK is developed by Java Software, a part of Sun Microsystems.

Java Software welcomes your comments and suggestions. You can contact us through email, or you can file a bug report. But please remember that Java Software does not offer free technical support. If you need help, try the More Technical Resources page, where you'll find links to FAQs, online tutorials, and many other resources.

Please file documentation bugs as described below. For other general comments about the documentation not targeted to any particular engineering team, please contact the Documentation Team.

File a Bug Report

You can use our bug tracking system to report bugs or request new features. Use our Bug Report Form.
