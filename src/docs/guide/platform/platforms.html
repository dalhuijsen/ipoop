Platforms

Overview

Sun provides implementations for the Java 2 SDK and Java Runtime Environment for Microsoft Windows, Linux, and the Solaris Operating Environment. See the installation notes for a given SDK or JRE release for information about which versions of these platforms are supported.

Other companies may provide implementations of the Java platform for other operating systems such as Macintosh, AIX, etc.
