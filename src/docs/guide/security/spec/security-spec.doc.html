JavaTM 2 Platform Security Architecture




Author: Li Gong

Version 1.2

This document gives an overview of the motivation of the major security features that are being implemented for JavaTM 2 Platform Security, introduces the new classes and their usage, discusses the impact of this new architecture on existing code, and gives thoughts on writing security-sensitive code in the future.
