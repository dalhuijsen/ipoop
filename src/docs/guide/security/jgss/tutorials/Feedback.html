Feedback

Please e-mail Java GSS-API and JAAS tutorial comments to: 

java-security@sun.com

Please specify which tutorial(s) you are referring to in your message.

Though we value your input, before sending problem reports please search the Java Security Q&A Archives:

http://archives.java.sun.com/archives/java-security.html

The above mailing list is not a subscription list or a support mechanism. It is simply a one-way channel that you can use to send comments to the Java 2 Standard Edition security team.

Please note that due to the volume of messages we receive, we may not be able to respond to every individual message.
